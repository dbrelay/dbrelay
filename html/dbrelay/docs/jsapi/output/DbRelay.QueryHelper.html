Class DbRelay.QueryHelper

Package:DbRelay
Defined In:sqlDbAccess.js
Class:QueryHelper
Extends:Object
Previously known as sqlDbAccess, this class provides an API for querying a single database. Usage: 
var qh = new DbRelay.QueryHelper({   
	sql_server : '99.99.99.99.99', 
	sql_port : '',  
	sql_database : 'mydatabase', 
	sql_user : 'myuser',                 
	sql_password : 'mypass',              
	connection_name : 'example2'
});
